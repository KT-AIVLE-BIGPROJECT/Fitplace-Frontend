## 💡 서비스 기획 배경
- 개인의 취향과 흥미를 중요하게 생각하는 사람들을 위해 개인의 취향을 반영한 맞춤형 장소를 추천해줄 뿐만 아니라 혼잡도도 같이 제공해주는 서비스를 제공하고자 하였습니다.

---
## ✏️ 서비스 설명
- **장소 추천**
  - 사용자의 정보(연령대, 성별, MBTI)와 취향 키워드(먹기,마시기,놀기 등)에 기반한 장소를 추천해줍니다.
- **인구 혼잡도 제공**
  - 서울 열린 데이터광장 홈페이지에 공개된 ‘서울 실시간 도시데이터’를 활용하여 실시간 혼잡도 확인 + 다음날 혼잡도 예측결과를 제공함으로써 장소 선택에 도움을 줍니다.

## 📊 사용 데이터
- **Kaggle MBTI Personality Types 500 Dataset**
  - https://www.kaggle.com/datasets/zeyadkhalid/mbti-personality-types-500-dataset
  - 장소들의 리뷰 데이터에 대해 MBTI를 분류하기 위해 사용
- **서울 실시간 도시데이터**
  - https://data.seoul.go.kr/SeoulRtd/
  - 서울 열린데이터 광장에서 제공하는 서울 실시간 데이터 open API로 1달간 10분 간격으로 데이터 수집
- **네이버 지도 장소 데이터**
  - 네이버 지도에서 [지역구 + 취향 키워드]를 조합한 검색 키워드(ex.강남구 한식) 검색 결과 크롤링 (Selenium, BeautifulSoup 사용)

## 🖥 개발 화면
- **메인화면 + 로그인**
  - [네이버리뷰 수 + 블로그리뷰 수]가 높은 인기있는 장소를 100개 중 7개를 랜덤하게 보여줌
  - 트위터 API를 사용하여 트위터 유저들이 많이 이용하는 맛집 태그로 검색 태그로 검색된 내용을 확인 가능
  - 로그인 시 사용자 프로필 입력 토대로 장소 추천 배너 생성
